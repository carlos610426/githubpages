HTML, CSS y JavaScript:

Son los tres pilares fundamentales del desarrollo web que trabajan juntos para
crear páginas web dinámicas e interactivas.
HTML se encarga de la estructura y el contenido, CSS del diseño y la presentación,
y JavaScript de la interactividad y funcionalidad.

Definiciones

HTML (HyperText Markup Language):

Es el lenguaje de marcado que define la estructura y el contenido de una página web.
Utiliza etiquetas para delimitar diferentes elementos, como párrafos, encabezados, imágenes, etc.

CSS (Cascading Style Sheets):

Es un lenguaje de estilos que se utiliza para controlar la presentación de una página web.
Permite definir colores, fuentes, márgenes, tamaños, etc.

JavaScript (Programming Language):

Es un lenguaje de programación que permite agregar funcionalidad interactiva a una página web.
Permite manipular elementos HTML y responder a eventos del usuario.

Analogía de la web y una casa:

HTML

Estructura de la casa.
Como las paredes, techos y distribución de habitaciones.
Define la estructura básica (títulos, párrafos, imágenes...).

CSS

Decoración y estilo de la casa: pintura, cortinas, muebles, etc.
Controla cómo se ve la página visualmente.

JavaScript

Sistemas eléctricos y funcionalidad: luces, puertas automáticas, sensores.
Añade interactividad y reacciones a eventos.

Resumen

"Los tres lenguajes son esenciales para el desarrollo web, y cada uno juega un papel crucial
en la creación de páginas web modernas y funcionales."
